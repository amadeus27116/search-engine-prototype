# search-engine-prototype

## Installation Guide

### Linux

#### Clone the repository
$ git clone https://github.com/amadeus27116/search-engine-prototype.git
$ cd search-engine-prototype

#### Create the virtual environment and install dependencies
$ python -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt

#### Start the app
$ python app.py

#### Note
The app will display a url along the lines of "http://127.0.0.1:<port-number>". To view the app, enter the url into
your browser to run the view the home page.

#### Deactivating the app
When finished running the app, type the following command to deactivate the virtual environment:
$ deactivate
